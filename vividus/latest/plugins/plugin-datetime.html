Plugin Date Time

General Info

Plugin provides functionality to generate, modify or validate the date.

Installation

Add dependency to your tests
implementation(group: 'org.vividus', name: 'vividus-plugin-visual', version: 0.2.0)

Properties

Required

There are now required properties for the plugin.

Optional

location.zone-id - Difens time zone which will be used. Default is set to a system default timezone.

Steps

Then the date '$date' is $comparisonRule current for $seconds seconds

Info

Validates the difference between current date and the date under test in seconds.

Parameters

  1. $date - the date in the ISO-8601 format

  2. $comparisonRule - Comparison Rule

  3. $seconds - the quantity of seconds between current and the date.

Usage

Step usage example
Then the date '2020-06-16T15:25:29.516580Z' is less than current for 10 seconds
